# FILE-HANDLING-UTILITY

## PROJECT OVERVIEW

This project demonstrates file handling operations in Java. The application performs three essential operations on text files:

1. Writing data to a file
2. Reading data from a file
3. Modifying (appending) data to an existing file

The project showcases Java's file input and output capabilities using FileWriter, FileReader, and BufferedReader classes.

## FEATURES

* Create and write text files
* Read file contents
* Modify existing files
* Exception handling using try-catch blocks
* Simple console-based interface

## TECHNOLOGIES USED

* Java
* FileWriter
* FileReader
* BufferedReader

## HOW TO RUN

1. Open the project in VS Code or any Java IDE.
2. Compile the program:

javac FileHandlingUtility.java

3. Run the program:

java FileHandlingUtility

4. Observe the file creation, reading, and modification operations.


## OUTPUT

The program:

* Writes text into sample.txt
* Reads and displays file contents
* Appends additional text
* Displays updated file contents

## CONCLUSION

This project successfully demonstrates file handling operations in Java, including reading, writing, and modifying text files while maintaining proper exception handling and documentation.
